  • Automatic Loading: The most recent backup is loaded automatically each night at 6 AM UTC. You can manually load different backups below.
  • Single Backup Model: Only one backup can be loaded at a time. Loading a new backup will restart the isolated test environment and replace the database (takes 1-2 hours).
  • Isolated Environment: Runs latest code but with real data from backups. Uses Docker technology to create a completely isolated copy of the system - changes here don't affect live Freegle.
  • Login: Only email/password authentication works. OAuth providers (Google, Facebook) are disabled since they're configured for production domains only.
  • Email Testing: All outbound email is captured in Mailhog (a local email testing tool) - nothing is sent externally, so it's safe to test email functionality.
